=====Software Requirements:=====

Eclipse Helios

Apache Tomcat 7.0

MySql

Microsoft Excel

HeidiSQL - SQL editor(optional)


=====Step to run the application:=====

--> run the .sql script (database dump) - the .sql file is present in the package iiitb.timesheet.util

--> import the application (.war file) 

--> check the connection settings in RuntimeSettings.java in the package iiitb.timesheet.util

	- check "databaseName"
	
	- check "dbUserID" 

	- check "dbPassword"

--> setup the server, specify the installation dirctory of the server.

--> build the application.

--> click on " run as - on server".
